Справка: Система поддержки принятия решений

Данная система предназначена для симуляции и анализа эффективности различных стратегий переработки сырья (сахарной свеклы) в условиях неопределенности. Цель — рекомендовать оператору наиболее устойчивую стратегию с наименьшими относительными потерями сахара.

Параметры симуляции

Система использует метод имитационного моделирования, где T — количество виртуальных экспериментов, а n — количество этапов переработки (недель).

  • Начальная сахаристость (α): Случайно генерируется в заданном диапазоне для каждой партии.
  • Коэффициент деградации (β): Определяет, насколько уменьшается (или, в случае дозаривания, увеличивается) доля сахара в партии с каждым этапом хранения.

Расчет эффективности

Для каждого эксперимента:

  1. Генерируется полная Матрица Состояний (C).
  2. Запускаются все реалистичные стратегии (greedy, thrifty и т.д.) для определения их результата.
  3. Определяется Оптимальный Результат (σopt) с использованием идеального алгоритма, который знает будущее (используется как эталон).
  4. Для каждой стратегии рассчитываются Относительные Потери (%) относительно этого идеального результата.

Рекомендация

Система предлагает стратегию, которая показала наименьшие усредненные относительные потери по итогам всей серии экспериментов.
